# Tamás Péter - Personal Portfolio & Graphics Blog

Welcome to the repository for my personal portfolio website! 

🌐 **Live Website:** [tamaspetii.github.io](https://tamaspetii.github.io)

## 👨‍💻 About Me
I am a Graphics Programmer and the creator of the **Synapse Engine**, a high-performance, GPU-driven rendering engine. My work focuses on modern rendering techniques, including mesh shader pipelines, hierarchical culling, and data-oriented Entity-Component-Systems (ECS) designed for extreme entity counts.

I am passionate about low-level optimization, real-time computer graphics, and sharing my knowledge with the community.

## 🚀 Tech Stack
This website is built with a minimalist, performance-focused architecture to beautifully showcase graphics projects and code snippets:
- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Hosting:** GitHub Pages (Automated CI/CD via GitHub Actions)
